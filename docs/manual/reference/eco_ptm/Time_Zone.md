# Time Zone

---

## Overview

The `Time_Zone` parameter defines the time zone where the simulation is performed.

---

## Example

```plaintext
Time_Zone
PST
End_Time_Zone
```

---

## Allowable entries

Allowable entries: PST (default), MST, CST, or EST

---

## Requirement

---

> ⚠️ **Warning:**  
> This parameter is required.
