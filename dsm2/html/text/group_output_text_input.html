GROUP_OUTPUT Section

Overview:

GROUP_OUTPUT is a section in the PTM text input that specifies DSS output to record residence of particles in group of water bodies.

Reference

Keyword Descriptions

GROUP
Name of the group whose members are augmented by the text line.  If the name does not match an existing group, a new group is created. Since the PTM uses only text input, the GROUP must be defined in the text  input file.
INTERVAL
Interval at which to record residence. 
B_PART
This is the name that will go in the B_PART of the output.
FILENAME
The name of the output file. If the file extension is *.dss, output is in DSS format. If the file extension is *.txt a text file output is produced.

Examples:

GROUP_OUTPUT
GROUPINTERVALB_PARTFILENAME
twitchell1HOURTWITCHELL${PTMOUTPUTFILE}
emmaton1HOUREMMATON${PTMOUTPUTFILE}
END
