TITLES Section

Overview:

TITLES are a text description of a model run that are used as headers in some text output. The title typically goes at the top of the text input file (hydro.inp, qual.inp, ptm.inp).

Reference

Keyword Descriptions

There are no keywords. Every line in the titles section is added to the title for the model run.


Examples:

The following example depends on the ENVVAR ${DSM2MODIFIER}. The advantage of this approach is that DSM2MODIFIER usually has to be altered meaningfully for each run as part of the simulation preparation process, so it is usually up-to-date. In contrast, it is hard to remember to change the TITLES section by hand.


TITLES
DSM2 DELTA HISTORIC SIMULATION: ${DSM2MODIFIER}
END
