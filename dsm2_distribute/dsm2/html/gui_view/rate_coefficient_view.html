Rate Coefficients

Overview:

Rate Coefficients are reaction and growth rates assigned to non-conservative constituents. This table assigns the rates to groups of water bodies (there usually aren't enough data to support individual assignments).

Rate Coefficient Table

The Rate Coefficient Table lists reaction rate coefficients for non-conservative constituents. Different rates can be assigned to different water bodies. The assignment is done using groups -- first you define a group and then you assign rate coefficients to the group. The rate coefficient is a top-level layered table, but it does not have a Use column because rate coeffients can be reassigned using higher layers.

Field Descriptions

Layer
Layer number on which the entry is defined.
Group Name
Name of the group to which the coefficient entry is assigned.
Constituent
Non-conservative constituent with which coefficient is associated.
Variable
Physical process governed by coefficient.
Value
Value assigned to the coefficient

Usage Notes:

  • Assignments on higher layers supercede assignments on lower layers, even if the patterns that cause the assignment are not the same.
  • All channels must have rate coefficients.
