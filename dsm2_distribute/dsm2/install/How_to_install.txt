DSM2_v8 installation instruction for Windows XP

1. Right click "My Computer", select "Propertites", 
   select the tab "Advanced", click the button "Environment Variables"

2. Delete dsm2 related values in variables "PATH" and "PYTHONPATH" 
   under both catogories "User varibales for XXX" and "System variables"
   
3. Run the installation file DSM2setup_8.0.2.exe
