PARTICLE_INSERTION

Overview:

Particle Insertion is a section in the PTM input that insertion of particles in water bodies over time.

Reference

Keyword Descriptions

NODE
The node at which the insertion is made.
NPARTS
Number of particles
DELAY
Delay before the first insertion.
DURATION
Interval over which insertion is distributed in time.
