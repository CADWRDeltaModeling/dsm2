PARTICLE_FLUX_OUTPUT

Overview:

PARTICLE_FLUX_OUTPUT is a section in the PTM text input that specifies DSS output to record movement of particles in group of water bodies.

Reference

Keyword Descriptions

B_PART
This is the name that will go in the B_PART of the output.
FROM_WB
Name of the water body or group that is the "from" location of theflux.
TO_WB
Name of the water body or group that is the "to" destination in the flux.
INTERVAL
Interval at which to record residence.
FILENAME
The name of the output file. If the file extension is *.dss, output is in DSS format. If the file extension is *.txt a text file output is produced. 

Examples:

FLUX_OUTPUT
NAMEFROM_WBTO_WBINTERVALFILENAME
TWITCHELLres:clifton_courtgroup:swp15MIN${PTMOUTPUTFILE}
EMMATONchan:216group:cvp15MIN${PTMOUTPUTFILE}
PAST_MTZchan:441group:mtz15MIN${PTMOUTPUTFILE}
WESTgroup:chipps_eastgroup:chipps_west15MIN${PTMOUTPUTFILE}
DIVERSION_AGgroup:allgroup:ag_div15MIN${PTMOUTPUTFILE}
END

Usage Notes:

  • Particle flux output can be in absolute number of particles or percentage of injection. The option is set by PTM_FLUX_PERCENT in the SCALAR section.
