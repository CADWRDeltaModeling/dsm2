Parameters

Overview:

Parameters are scalar model variables used to specify model-wide numerical properties and echoed output levels. They are the equivalent of the text input SCALAR section. All of the parameters are interpreted as text and can be replace by ENVVARS.

Parameter Table

The Parameter table comprises name-value pairs for scalars. The scalars that are allowed depend on the specific model. The table is a top-level, layered table but does not have the typical Use column because parameters typically have defaults and cannot be "removed".

Field Descriptions

Layer
Layer number on which the entry is defined.
Parameter Name
Name of the parameter. This is the identifier of the parameter.
Description
Long description of the parameter.
Value
Value assigned to the parameter. These are interpreted by the model first as text (to allow substitution using ENVVARS) and then converted to the correct data type and validated.
