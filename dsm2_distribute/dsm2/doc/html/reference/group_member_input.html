GROUP_MEMBER Section

GROUP_MEMBER is a section that allows the user to describe groups of model objects, for instance groups of water bodies or groups of boundary inputs. The names of groups must be declared in GROUP section before they are defined in this section.

Reference

Keyword Descriptions

GROUP_NAME
Name of a group whose members are augmented by the text line. If a group with the name is defined already, the members in the new definition will be added into the previously-defined group.
MEMBER_TYPE
Type of model object included in the group. 
PATTERN
A regular expression that matches the identifier of the members.

Examples:

GROUP_MEMBER
GROUP_NAMEMEMBER_TYPEPATTERN
twitchellchannel4[5-8]# matches channels 45-48
twitchellchannel(83|280|310)# matches three channels
emmatonchannelrange:430-434# no spaces allowed
ag_divqextdicu_div_.*# dot-star is a wildcard
ag_divqextbbid.*
swpqextswp
mtzstagemtz.*# matches stage input name
END
