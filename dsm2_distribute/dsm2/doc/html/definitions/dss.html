HEC Data Storage System (DSS)

The Hydrologic Engineering Center Data Storage System (HEC-DSS or just DSS) is a binary storage format for time series data developed by the Army Corps of Engineers Hydrologic Engineering Center (HEC). All time-varying input data for DSM2 is specified in DSS format, and it is also an important output format for the models.

In DSS, time series are stored and retrieved using a data reference called a "path". The path is a six part name seperated by forward slashes and lettered sequentially: /A/B/C/D/E/F/. For example, historical stage data for Martinez might be stored under the path:

/HIST+CHAN/RSAC054/STAGE/01JAN1980/15MIN/DWR-D1641/

In DSM2, path parts are typically assigned the following values:

A PART
Generator or instrument location.
B PART
Location name, node number, or channel number.
C PART
Variable observed.
D PART
The starting date for the time series data or time period covered.
E PART
Time interval of time series data.
F PART
Study name and scenario, agency and/or program information.
