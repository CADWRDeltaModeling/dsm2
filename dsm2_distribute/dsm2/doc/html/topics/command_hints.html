Command Window Hints

The Open Command Window Here toy is easy to find on the web. It is much faster than starting a command window from the start menu and using the cd command. At DWR, this tool is kept in \\delta-mod\ydrive\softwareupdates\ CommandPromptHere.


Setting Reasonable Screen Buffer Size:

1. Click on the Command Icon in the top left corner of your command window. A drop down menu will appear. Select Properties.

 

2. In the Layout tab, enter the following values: 

Width: 400 

Reason: This is large enough to capture any single line DSM2 will write)

 Height: 9999:

Reason: This is the maximum. Retains as long as possible a history that can be scrolled.

Width: 90
Reason this is a preference that works well with large modern monitors.

 Height: 48
Reason: This is a preference that works well with large modern monitors.

3. Click OK. In the final dialog box, select “Save properties for future windows”.

Use Autocompletion

When you start to type names of a file or directory in the command prompt, type the few letters then tab – it will complete the name for you. If there is more than one possible completion, tab will alternate between choices.

Cutting Text from the Command Line:

This is convenient if you are reporting a bug (far preferred to screen shots) or need to manipulate or search your output.

1. To start the selection, you can click on the Command Icon in the top left corner of your command window and choose Edit >Mark. Alternatively, you can right click anywhere on the command window and select Mark in the context menu that will appear.

 

 
2. Select a block of text. Selected text will turn white. When you complete your selection, copy it to the clipboard by pressing return or selecting Edit > Copy from the top left Command Window menu.

 

 3. You can paste the copied text anywhere in the normal way (e.g., email, Word document). You can paste into the command window using the right-click context menu.
