Briefly,
	Start vscript by typing vscript on the prompt. Then to execute
the contents of any file ending in .py extension type. Report bugs to
me: nsandhu@osp.water.ca.gov

>>> execfile('file.py')


Directory structure

class/:
	These are notes from classes that I have given on vscript &
jpython 

examples/:
	These are example scripts that may help one understand some of
the concepts involved

production/:
	These are scripts that are used to do meaningful tasks. As
these scripts graduate they are placed into vutils or Java classes

test-scripts/:
	These are test scripts used to test the sanity of the release.

testdata/:
	The testdata used by the example and test-scripts code
